## Rules for writing your questions

- Number each question Q1 through to Q10
- Always provide exactly four options: A) B) C) D)
- The Answer: line must use just the letter — e.g. `Answer: B`
- No blank lines *within* a question block — only between questions
- No special characters in question text (avoid &, %, #, etc.)
- Keep questions to one line if possible — long questions may wrap oddly on phones

## Tips

- Save from Word or Google Docs as **PDF**, not as a .docx
- Use a plain font (Arial, Calibri, Times New Roman) — decorative fonts can cause parsing errors
- Do a test upload with 3 questions before your full set
